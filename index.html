Mi vida académica.

Te cuento que mi educación básica y media fue cursada en un pequeño establecimiento educacional llamado Colegio General Velásquez Bórquez del pequeño pero adorable pueblo de Puchuncaví.

Después de tener trabajos esporádicos di el salto para estudiar lo que ahora es mi pasión, la programación y así fue como me titulé de “Analista Programador” de Inacap Valparaíso con el logro de ser número 1 del ranking de egreso.

La gran sorpresa y guinda de la torta vendría después de obtener un poco más de dos años de experiencia, ya que, una simple postulación a una beca del Mineduc me permitió realizar un curso de especialización en Madrid, España, específicamente en la Universidad de La Salle International Graduate School, momento en el cual obtuve 3 certificaciones en Microsoft, las cuales son:

  • MTA: Software Development Fundamentals - Certified 2020.
  • Microsoft Certified: Azure Fundamentals.
  • Microsoft Certified: Azure Administrator Associate.





Experiencia Laboral

Al terminar mis estudios de Analista Programador realice mi practica profesional en la Municipalidad de Puchuncaví, donde quede contratado de manera inmediata. Dentro de las funciones que he cumplido en el municipio la principal es la innovación, las tareas más significativas realizadas son las siguientes:

  • Desarrollo de software de gestión de entrega de beneficios gubernamentales y de denuncias medioambientales, los cuales han ayudado a mantener un orden y obtener información para la toma de decisiones con todos los datos recopilados.
  • Desarrollo de software de escritorio para creación de nóminas del sistema web de gobierno “SEM GOB” interconectado con las bases de datos del software en arriendo de CAS CHILE, permitiendo que ambos datos se reflejen en ambas plataformas y mejorando considerablemente los tiempos de respuesta a las solicitudes de contribuyentes permitiendo reducir el tiempo de carga de estas nóminas de 20 vehículos por hora a 90 vehículos por hora, lo que conllevo a una mayor carga de vehículos, por ende una mayor recaudación para la institución.
  • Escribir y evaluar licitaciones públicas para la adquisición o contratación de equipos y/o servicios
  • Desarrollo de un nuevo sitio web municipal acorde a las exigencias de los contribuyentes

Las principales tecnologías utilizadas son:

  • Azure
  • SQL Server
  • C#
  • Php
  • HTML 5
  • CSS 3
  • Microsoft Access
